Network-Optimised Spiking (NOS)

NOS is a compact two-state spiking model for event-driven networking. The state v represents normalised queue occupancy and u captures recovery or resource. This page links to the interactive demos and the experiment code.

Paper: arXiv:2509.23516 (doi: 10.48550/arXiv.2509.23516)
